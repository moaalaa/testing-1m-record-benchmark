# How to run benchmarks

1. Place `test-file.csv` in project root
2. Navigate to the desired folder, e.g., `mysql/boring/a_plain/`
3. Run the language-specific script:
   - PHP: `php plain.php`
   - Node.js: `node plain.js`
   - Python: `python plain.py`
   - Go: `go run plain.go`
   - Rust: `cargo run --bin plain`
   - Dart: `dart plain.dart`
4. Log metrics (time, rows/sec, memory, CPU) to `results/`
